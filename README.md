# where-did-it-flow

Monorepo scaffold for a local-first money flow app.

## Workspaces

- `fe`: Vite + React + TypeScript frontend scaffold
- `be`: Fastify + SQLite + Drizzle backend scaffold

## Quick start

1. Install deps:

```bash
npm install
```

2. Run FE + BE:

```bash
npm run dev
```

3. Generate/migrate database schema:

```bash
npm run db:generate
npm run db:migrate
```
